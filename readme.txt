Here are the compiler names/setups needed for Code::Blocks
GNU GCC Compiler=GDB, MINGW
msvc2013-x86=CDB, WIN8&VS
msvc2013-x64=CDB64, WIN8&VS
Some people might not work out the variables for themselves so here are some
examples:
IUP=C:\Users\Public\Github\iup3.10.1
IUP64=C:\Users\Public\Github\iup3.10.1-x64
IPIPE=C:\Users\Public\Github\ipipe
VS=C:\Program Files (x86)\Visual Studio 2013
WIN8=C:\Program Files (x86)\Windows Kits\8.1
BIN=C:\Program Files (x86)\Compiled Libs
LIB=C:\Program Files (x86)\Compiled Libs
MINGW=C:\sdk\mingw
